# RecipeBlog
### Api and site for recipes
---
# How to run this app

To run the app locally:
1. Clone this repo to your system
2. Open repository folder in Visual Studio Code
3. Run command `npm i`
4. Once packages have installed, in the "backend" folder, run the command `node .`
5. ..
6. Profit - your API is running at address `localhost:9090`
